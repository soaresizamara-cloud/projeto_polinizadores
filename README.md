# # 🐝 Projeto Polinizadores

Site educacional interativo desenvolvido para conscientizar sobre a importância das abelhas na produção de alimentos, na biodiversidade e na preservação dos ecossistemas.

## 📖 Sobre o Projeto

As abelhas desempenham um papel fundamental na polinização de diversas culturas agrícolas. Este site apresenta informações educativas, conteúdos multimídia e atividades interativas para demonstrar como esses polinizadores contribuem para a segurança alimentar mundial.

## ✨ Funcionalidades

- 🖼️ Galeria de imagens temáticas
- 🎯 Quiz interativo sobre polinização
- 📊 Indicadores informativos
- 📑 Abas de conteúdo dinâmico
- 🎠 Carrossel de exemplos e casos reais
- 🎥 Vídeo educativo incorporado
- ❓ Seção de Perguntas Frequentes (FAQ)
- ♿ Recursos de acessibilidade
  - Aumento e redução de fonte
  - Alto contraste
  - Navegação facilitada
- 📱 Layout responsivo para dispositivos móveis

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6)

## 🎯 Objetivo

Promover a educação ambiental e conscientizar a população sobre a importância das abelhas para a produção de alimentos, a manutenção da biodiversidade e o equilíbrio dos ecossistemas.

## 📂 Estrutura do Projeto

```text
├── index.html
├── style.css
└── script.js
```

## 🚀 Como Executar

1. Faça o download dos arquivos do projeto.
2. Mantenha os arquivos na mesma pasta.
3. Abra o arquivo `index.html` em qualquer navegador moderno.
